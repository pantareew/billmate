# BillMate
## Overview
This app is a smart billâ€‘splitting platform built to make managing shared expenses effortless. It removes the hassle of manual calculations by automating receipt extraction, debt tracking, and balance management. Users simply upload a receipt, choose a group, and the app takes care of splitting costs, updating payment statuses, and keeping everything transparent.

## Features
* AI extract receipts automatically (totals, merchants)
* Split bills instantly across groups
* Real-time debt tracking (who owes who)
* View clear payment status for each bill
* Dashboard summary of unpaid and pending balances
* Debtors can upload receipts for creditor approval
* Get notified when added to a shared bill

## Technologies
* Frontend: Next.js (App Router), TypeScript, Tailwind CSS
* Backend: FastAPI (Python)
* Database: PostgreSQL
* Auth & Storage: Supabase
* AI: OpenAI Vision
